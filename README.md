# ⏱️ Focus Timer

Odaklanmayı artırmak ve verimliliği en üst düzeye çıkarmak için tasarlanmış, açık kaynaklı, minimalist bir Pomodoro sayacı.

## 🌟 Özellikler
* **Pomodoro Tekniği:** Standart 25 dakikalık çalışma ve odaklanma süreleri.
* **Kullanım Kolaylığı:** Tek tıkla başlatma, duraklatma ve sıfırlama.
* **Kurulumsuz:** Tamamen tarayıcı üzerinden çalışır, herhangi bir kurulum gerektirmez.
* **Modern Tasarım:** Göz yormayan, karanlık tema (Dark Mode) odaklı şık arayüz.

Bu proje hiçbir dış kütüphane veya framework kullanılmadan, tamamen temel web teknolojileri ile geliştirilmiştir:
